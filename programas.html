Calculadora Simples

Este programa funciona como uma calculadora, realizando operações de adição, subtração, multiplicação e divisão.


# Definindo os números
numero1 <- 10
numero2 <- 5

# Adição
adicao <- numero1 + numero2
print(paste("Adição:", adicao))

# Subtração
subtracao <- numero1 - numero2
print(paste("Subtração:", subtracao))

# Multiplicação
multiplicacao <- numero1 * numero2
print(paste("Multiplicação:", multiplicacao))

# Divisão
divisao <- numero1 / numero2
print(paste("Divisão:", divisao))

Explicação: Definimos dois números, numero1 e numero2. Em seguida, realizamos as operações de adição, subtração, multiplicação e divisão, armazenando os resultados em variáveis separadas. Finalmente, usamos a função print() para exibir os resultados de cada operação.
